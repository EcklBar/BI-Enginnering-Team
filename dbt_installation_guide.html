BigQuery Configuration
Setup Google Cloud authentication
Pending
▶
Configure dbt to work with BigQuery by installing Google Cloud CLI and setting up appropriate authentication credentials.
1
Install Google Cloud CLI
Verify that the gcloud command is available on your system. If not installed, download from the link below:
Download Link
https://cloud.google.com/sdk/docs/install
🔍 Hardware Detection
To determine your machine architecture, run: uname -m
2
Authenticate with Google Cloud
Set up application-default credentials with the required scopes for BigQuery access:
Authentication Command
gcloud auth application-default login \
--scopes=https://www.googleapis.com/auth/bigquery,\
https://www.googleapis.com/auth/drive.readonly,\
https://www.googleapis.com/auth/iam.test
⚠️ Important Note
Remove all spaces from the command when copying to your terminal for proper execution.

Authentication Workflow:

  1. Verify gcloud is installed: gcloud --version
  2. Execute the authentication command above
  3. Browser will open for Google account verification
  4. Sign in with your authorized organizational Google account
  5. Confirm successful authentication message
  6. Verify setup with: gcloud auth list

Install Command
pip install dbt-bigquery
2
Initialize dbt Project
Initialize your dbt project using the official init command:
Init Documentation
https://docs.getdbt.com/reference/commands/init
Initialization Command
dbt init
📋 dbt init Configuration Prompts
When you run dbt init, follow these exact inputs to match the organizational configuration:
3
Configure dbt init Prompts

Answer the prompts as follows:

  1. Project name: dbt_bi
  2. Database (adapter): Select bigquery from the list
  3. Authentication method: Select oauth
  4. Project ID: bg-outbrain-bi
  5. Dataset (schema): mart_dev
  6. Threads: 64
  7. Job timeout (seconds): 3000
  8. Location: US
  9. Priority: interactive
⚠️ Expected Output
After completing the configuration, your profiles.yml should contain the following structure:
Expected profiles.yml Configuration
dbt_bi:
  outputs:
    dev:
      database: bg-outbrain-bi
      location: US
      method: oauth
      priority: interactive
      schema: mart_dev
      threads: 64
      timeout_seconds: 3000
      type: bigquery
  target: dev

Setup Workflow:

  1. Ensure your virtual environment is still active: (dbt_venv)
  2. Install dbt with BigQuery adapter using the command above
  3. Navigate to your project directory if not already there
  4. Run dbt init to start the configuration wizard
  5. Follow the configuration prompts exactly as specified above
  6. Verify your profiles.yml matches the expected output
  7. Test the connection with: dbt debug
🔍 Verification
After setup, you can verify your configuration by running dbt debug to ensure all connections are working properly.
🎉 Congratulations! You have successfully completed the dbt installation setup. Your development environment is now ready for data transformation workflows.
